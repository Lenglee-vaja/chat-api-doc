## 1. Get All Conversations

Retrieves a paginated list of conversations for a specific user.

#### Request

| Method | Endpoint |
|--------|----------|
| GET    | `https://chat-api.lailaolab.com/api/conversations` |

#### Query Parameters

| Parameter | Type    | Required | Description |
|-----------|---------|----------|-------------|
| userId    | String  | Yes      | ID of the user fetching conversations |
| skip      | Number  | No       | Number of records to skip (default: 0) |
| limit     | Number  | No       | Maximum number of records to return (default: 30) |

#### Request Headers

```json
{
    "Authorization": "Bearer {your_access_token}"
}
```
