# Register User

This endpoint allows you to register a new user in the system. Once registered, users can authenticate and interact with the Chat API using the **Secret Key** provided by the admin.

## API Endpoint

| Method | Endpoint |
|--------|----------|
| POST   | https://chat-api.lailaolab.com/api/users |

## Request Header

```json
{
    "Authorization": "Bearer {secret_key}"
}
```

## Request Body

```json
{
    "platformUserId": "671a18832572dd3242ea2455",
    "fullName": "John Doe",
    "phone": "081234567890",
    "email": "john.doe@example.com",
    "profileUrl": "https://example.com/profile.jpg"
}
```

| Field name | Data type | Required | Description |
|------------|-----------|----------|-------------|
| platformUserId | String | Yes | Unique identifier for the user |
| fullName | String | No | Full name of the user |
| phone | String | No | Phone number of the user |
| email | String | No | Email of the user |
| profileUrl | String | No | URL of any image Profile |
