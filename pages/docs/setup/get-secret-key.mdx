## Get Secret Key

In this step, the **admin** will create the **API Secret Key** for a customer and send it securely. The customer will use this key for authentication when making requests to the Chat API.

> **Important**: The API key grants access to the Chat API. It should be treated as sensitive information and not shared publicly.
